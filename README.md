# 🌦️ Weather App

A full-stack weather application built with React, Node.js, Express, and OpenWeather API. Users can search for any city, view current weather conditions, check a 5-day forecast, and get weather data based on their current location.

## 🚀 Features

* Search weather by city name
* Current location weather using Geolocation API
* 5-day weather forecast
* Recent search history (stored in localStorage)
* Responsive and clean UI
* Loading states and error handling
* Full-stack architecture with separate frontend and backend
* Deployed using Vercel (Frontend) and Render (Backend)

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/komalahuja0/Weather-App.git
cd Weather-App
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm start
```
